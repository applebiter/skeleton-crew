# Vosk Speech Recognition Models

Place your downloaded Vosk models in this directory.

## Quick Setup

```bash
cd /home/sysadmin/Programs/skeleton-crew/models/vosk
wget https://alphacephei.com/vosk/models/vosk-model-en-us-0.22.zip
unzip vosk-model-en-us-0.22.zip
rm vosk-model-en-us-0.22.zip
```

The application will automatically detect models in this directory.

For more information, see the main models/README.md file.
